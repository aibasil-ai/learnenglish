# LearnEnglish

LearnEnglish 是一個以遊戲化方式學習英文單字的 Web App，聚焦新聞常見詞彙。

## 主要功能

- 100 個關卡（20 類主題，每類 5 關）
- 每個主題第 1 關預設解鎖，可自由選類別學習
- 單字卡翻面學習、單字與例句語音播放
- 4 種測驗模式：聽力、字義、認字、拼字
- 測驗答對/答錯音效回饋
- 深色模式與發音口音設定自動儲存

## 技術棧

- React 18
- Vite
- Tailwind CSS
- Web Speech API
- Web Audio API

## 本機開發

```bash
npm install
npm run dev
```

預設開發網址：`http://localhost:5173`

## 建置

```bash
npm run build
```
